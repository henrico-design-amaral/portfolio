\## 1. Objetivo deste documento



Este documento define como ChatGPT, Antigravity, Codex local e demais ferramentas de IA devem trabalhar juntas dentro do projeto Portfolio.



A função deste arquivo é evitar:



\- perda de contexto;

\- retrabalho;

\- decisões contraditórias;

\- alterações simultâneas sem controle;

\- mudanças visuais aleatórias;

\- quebra de estrutura consolidada;

\- instalação desnecessária de ferramentas;

\- dependências entre agentes sem documentação.



Este arquivo não substitui os arquivos técnicos do projeto.  

Ele define o comportamento esperado dos agentes.

# Project Documentation

This folder contains the portfolio project documentation.

## Main sections

- `project/` — project control, tasks, handoff and product notes.
- `governance/` — rules, quality gates, accessibility and constitution.
- `orchestration/` — AI workflow and agent operating model.
- `setup/` — local environment and setup notes.
- `decisions/` — decision log.
- `logs/` — session logs.
- `design/` — design direction and visual documentation.
- `audits/` — performance, accessibility and repository audits.

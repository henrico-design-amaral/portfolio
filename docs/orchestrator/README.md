# Orchestrator — Portfolio

Esta pasta documenta como o Portfolio usa a Fase 1 do Henrico Agent OS.

## Objetivo

Conectar o Portfolio ao orquestrador global localizado em:

`C:/Users/henri/Documents/!PROJETOS/00_SYSTEM/henrico-agent-os`

## Arquivos locais

- `HANDOFF.md`
- `TASKS.md`
- `DECISIONS.md`
- `QUALITY_GATES.md`
- `docs/orchestrator/PHASE_1.md`

## Regra central

Nenhuma IA é fonte da verdade.

A fonte da verdade é o contexto versionável em Markdown.

## Uso prático

Antes de qualquer alteração relevante no Portfolio, agentes devem ler:

1. `PROJECT_CONTROL.md`
2. `AGENTS.md`
3. `GEMINI.md`
4. `HANDOFF.md`
5. `TASKS.md`
6. `DECISIONS.md`
7. `QUALITY_GATES.md`

Depois devem:

1. verificar branch atual
2. verificar `git status`
3. propor plano
4. aguardar aprovação quando necessário
5. executar alteração mínima
6. testar
7. sugerir commit

# Design Taste Review Skill

## Purpose

Review visual quality, hierarchy and refinement without redesigning the project unnecessarily.

## Checklist

- hierarchy;
- spacing;
- typography scale;
- density;
- contrast;
- section rhythm;
- editorial quality;
- consistency with current portfolio language;
- unnecessary decoration;
- visual clichés;
- mobile composition.

## Rule

Do not replace the existing visual identity.

Improve taste by reducing noise, sharpening hierarchy and preserving the current direction.

## Output

Report:
- visual strengths;
- visual weaknesses;
- specific corrections;
- what should not be changed.

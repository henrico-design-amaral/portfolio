# External Reference Research Skill

## Purpose

Use external references to inform portfolio decisions without copying them.

## Use cases

- analyze strong portfolio pages;
- inspect case-study structures;
- extract design system documentation patterns;
- compare editorial portfolio language;
- collect reference principles.

## MCP support

Use Firecrawl MCP when available for external page extraction and research.

Use Playwright MCP when available for visual inspection of local or live pages.

## Rules

- Do not copy proprietary layouts or text.
- Extract principles, not assets.
- Cite sources in research notes when applicable.
- Keep recommendations relevant to this portfolio.

## Output

Report:
- source inspected;
- useful principles;
- what applies to the portfolio;
- what should be ignored.

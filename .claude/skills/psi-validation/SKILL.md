# PSI Validation Skill

## Purpose

Run and document PageSpeed Insights validation.

## Procedure

1. Create an audit folder under `docs/audits/performance/`.
2. Run PSI mobile.
3. Run PSI desktop.
4. Save JSON outputs.
5. Create README with scores, diagnostics and comparison to previous baseline.
6. Do not alter product files in validation branches.

## Rule

Validation branches are documentation-only.

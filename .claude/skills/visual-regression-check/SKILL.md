# Visual Regression Check Skill

## Purpose

Perform a basic visual sanity check before committing UI changes.

## Checklist

- Hero renders.
- Navigation desktop works.
- Navigation mobile works.
- Sections keep spacing and hierarchy.
- Cards render.
- Modals work when applicable.
- PT/EN still works when applicable.
- No obvious horizontal scroll.
- No broken image paths.
- No console-breaking JS changes.

## Rule

Do not mark visual work as complete based only on code diff.

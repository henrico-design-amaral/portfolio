# Henrico Amaral Portfolio

Static portfolio for decision-infrastructure work across complex enterprise platforms.

## Structure

- `index.html` - public portfolio page.
- `assets/css/site.css` - local styles and font declarations.
- `assets/js/site.js` - small interaction layer for the mobile menu.
- `assets/fonts/` - local font files.
- `docs/` - case and product documentation.
- `docs/design-system*.html` - preserved design system studies.
- `case-references/` - local raw case references; private material, not public deploy content.

The site is intentionally static and published through GitHub Pages from `main`.

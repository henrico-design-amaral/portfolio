# AI MEMORY — Portfolio_v2

Esta pasta contém a memória persistente específica do projeto Portfolio_v2.

Ela registra decisões, contexto, padrões, restrições, problemas, mudanças e aprendizados relacionados exclusivamente ao portfólio profissional atual de Henrico Amaral.

Esta memória não substitui a memória global do workspace em:

../ia-memory

A memória global orienta o modo de trabalho.
Esta memória local registra as decisões específicas do portfólio.

## Objetivo desta memória

Evitar:

- perda de contexto
- retrabalho
- decisões contraditórias
- mistura com outros projetos
- alterações visuais isoladas
- textos genéricos
- repetição de soluções rejeitadas
- commits com escopo errado

Garantir:

- continuidade
- coerência visual
- coerência textual
- rastreabilidade
- evolução controlada
- memória de decisões
- alinhamento com o posicionamento profissional de Henrico

## Arquivos

- 00-global-profile.md: perfil profissional relevante para este portfólio
- 01-project-context.md: contexto geral do projeto
- 02-product-positioning.md: posicionamento estratégico do portfólio
- 03-design-system.md: memória do sistema visual
- 04-content-style.md: regras de escrita e tom
- 05-technical-decisions.md: decisões técnicas
- 06-visual-decisions.md: decisões visuais específicas
- 07-open-issues.md: problemas pendentes
- 08-changelog.md: histórico de mudanças
- 09-do-not-repeat.md: soluções rejeitadas ou proibidas

## Regra de uso

Antes de qualquer alteração relevante:

1. Ler este README.
2. Ler os arquivos de memória relevantes.
3. Analisar o código real.
4. Executar apenas o necessário.
5. Atualizar a memória após decisões relevantes.

## Regra final

A memória deste projeto é parte da infraestrutura do portfólio.

Não é documentação auxiliar.

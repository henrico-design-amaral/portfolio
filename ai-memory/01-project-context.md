# PROJECT CONTEXT — Portfolio

Portfolio é o portfólio profissional atual de Henrico Amaral.

Este é o projeto ativo principal no workspace !PROJETOS.

## Objetivo

Criar um portfólio forte, claro e estratégico para reposicionar Henrico no mercado como Senior Product Designer especializado em sistemas complexos, plataformas enterprise, arquitetura de informação, visualização operacional e design de interfaces para decisão.

O objetivo não é apenas apresentar trabalhos anteriores.

O objetivo é construir uma narrativa profissional mais forte, capaz de diferenciar Henrico de designers genéricos e aproximá-lo de oportunidades mais alinhadas ao seu perfil real.

## Função do site

O site deve funcionar como:

- portfólio profissional
- peça de posicionamento
- prova de maturidade visual
- prova de pensamento sistêmico
- estrutura narrativa dos principais cases
- vitrine técnica e estratégica
- base para abordagem em LinkedIn, candidaturas e conversas com recrutadores

## Seções principais

As principais seções do portfólio são:

- Hero
- Método
- Cases
- Detalhes dos cases
- CTA final
- Possíveis páginas internas de case
- Possível design system ou documentação visual

## Cases principais

### Petrobras / SALA CAR

Case relacionado a operações críticas offshore, ocorrências, resposta operacional, monitoramento, coordenação de recursos e visualização de decisão.

Deve comunicar:

- complexidade operacional
- criticidade
- coordenação
- geografia operacional
- tomada de decisão
- sistemas internos
- clareza em ambiente de pressão

### Bayer

Case relacionado a dados, agricultura, interpretação, operação técnica e suporte à decisão.

Deve comunicar:

- dados aplicados
- leitura contextual
- interpretação de informação
- suporte ao usuário técnico
- organização de fluxos de análise

### Ambev

Case relacionado a logística, operação, monitoramento e acompanhamento de processos.

Deve comunicar:

- fluxo operacional
- escala
- monitoramento
- eficiência
- clareza de status
- coordenação entre atores

### BMG Conta PJ

Case relacionado a produto financeiro para empresas, abertura de conta, serviços modulares, fluxos de validação e operação bancária.

Deve comunicar:

- produto digital
- jornada financeira
- modularidade
- confiança
- estruturação de serviços
- clareza para pequenas empresas

## Estado atual

Este projeto está em evolução ativa.

As seções Hero, Método e Cases são áreas críticas de consistência visual e narrativa.

Qualquer ajuste visual deve comparar a seção alterada com o sistema geral do site.

## Regra de continuidade

Nunca alterar uma seção isoladamente sem avaliar impacto sobre:

- Hero
- Método
- Cases
- backgrounds
- motion
- grid
- tipografia
- espaçamento
- tom textual
- responsividade

## Critério de sucesso

O portfólio deve parecer:

- maduro
- técnico
- editorial
- preciso
- estratégico
- específico
- autoral
- sofisticado sem excesso
- difícil de confundir com template
